# IOTApp

Executanto o app.
Primeiro, baixo o App Expo Go.

Instale todas as bibliotecas e pendências no seu computador que forem necessárias para rodar o código. UTILIZE O COMANDO YARN PARA INSTALAR!!! NÃO UTILIZE NPM!!!!

Após instalar tudo, rode o programa com o comando yarn start e espere o código carregar.
Escaneie o QRCode gerado no terminal, utilizando o app Expo Go no seu celular.

Pronto. É só isso.
